    Class: Instance

    Defined in: cloudbrowser.coffee

    Overview

    CloudBrowser application instances a.k.a. virtual browsers.

    Instance Variables

    id - The (hash) ID of the instance.
    name - The name of the instance.
    dateCreated - The date of creation of the instance.
    owners - The owners of the instance.
    collaborators - The users that can read and write to the instance.
