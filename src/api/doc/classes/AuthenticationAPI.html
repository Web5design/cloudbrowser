    Instance Method Details

    - (void) logout()

    Logs out all users connected to this application instance.

    - (void) login(user, password, callback)

    Logs a user into this CloudBrowser application. The password is hashed using pbkdf2.

    Parameters:

    • user (User) — The user that is trying to log in.
    • password (String) — The user supplied plaintext password.
    • callback (Function) — A boolean indicating the success/failure of the process is passed as an argument.

    - (void) googleLogin()

    Logs a user into the application through their gmail ID.

    - (void) sendEmail(toEmailID, subject, message, callback)

    Sends an email to the specified user.

    Parameters:

    • toEmailID (String) — The email ID of the user to whom the message must be sent.
    • subject (String) — The subject of the email.
    • message (String) — The content of the email.
    • callback (Function) — No arguments are passed to the callback.

    - (void) signup(user, password, callback)

    Registers a user with the application and sends a confirmation email to the user's registered email ID. The email ID is not activated until it has been confirmed by the user.

    Parameters:

    • user (User) — The user that is trying to log in.
    • password (String) — The user supplied plaintext password.
    • callback (Function) — No arguments are supplied.

    - (void) sendResetLink(callback)

    Sends a password reset link to the user at their registered email ID.

    Parameters:

    • callback (Function) — false is passed as an argument if the user is not registered with the application else, true is passed.

    - (void) getResetEmail(queryString)

    Gets the user's email ID from the url

    Parameters:

    • queryString (String) — Must be the location.search of the instance.

    - (void) resetPassword(queryString, password, callback)

    Resets the password for a valid user request.

    Parameters:

    • queryString (String) — Must be the location.search string of the instance.
    • password (String) — The new plaintext password provided by the user.
    • callback (Function) — A boolean is passed as an argument to indicate success/failure.
